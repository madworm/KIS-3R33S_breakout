
* D2 on the output side MUST be removed!

  Failure to do so will cause it to fail shorted to GND
  sooner or later.

* Order of soldering parts:

  1) LED + resistor
  2) DC-DC module
  3) Electrolytic capacitor
  4) 10-turn potentiometer
  5) Terminal blocks

* Required tools:

  - Flux
  - Tweezers
  - Solder braid
  - BEEFY soldering iron (lots of copper on board)
